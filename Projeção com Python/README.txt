Exercício de Projeção para Verificação do Término de Backlog
Este projeto analisa o backlog de chamados em um determinado contexto, utilizando as bibliotecas NumPy, Pandas e Matplotlib para realizar os cálculos necessários.

Objetivos
Neste estudo, busco calcular:

Média de Chamados em Aberto: Análise da quantidade média de chamados pendentes.
Média de Chamados Fechados: Avaliação da quantidade média de chamados resolvidos.
Taxa Líquida de Resolução: Cálculo da eficiência na resolução de chamados.
Tempo Estimado para Zeragem do Backlog: Projeção de quanto tempo levará para que o backlog seja completamente resolvido.

#Teste de Hipótese
Neste projeto, formulo uma hipótese para avaliar a possibilidade de reduzir em um mês o tempo necessário para zerar o backlog. 
A hipótese analisada é:
Aumento da Taxa Líquida de Resolução: Se conseguirmos elevar nossa taxa líquida de resolução para 33, será viável reduzir o tempo para zerar o backlog em um mês.

###Conclusão###
Com base nos resultados obtidos durante a análise, é seguro afirmar que, se elevássemos a nossa taxa líquida de resolução para 33, 
conseguiríamos reduzir o tempo estimado para zerar o backlog em um mês.
